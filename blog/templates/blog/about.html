IDEs I Use/Used For Programming

  1. Eclipse

    2. Eclipse is nowadays my main IDE that I use while programming with Java. The main thing I love about Eclipse is that it has great tools designed for debugging and running code, also it has great tool for git integration and it allows great project management inside itself.

  2. Visual Studio Code

    3. Visual Studio is a light-weight text editor which allows us to write code in different kind of programming languages. While it is not as powerfull as an Integrated Development Environment(IDE), it can become as powerful with the help of it's plugins.

  3. Vim

    4. Vim is nowadays my main text editor I use while developing non-Java softwares such as automation scripts and/or web applications. I also use Vim in my university studies when I study different programming languages such as Haskell and C or whenever I need to connect and write code over school labs and virtual machines. The thing I love about vim is that it comes pre-installed on most of the Unix based operating systems and because of it's minimalistic design, it is very light weight. However, Vim also proves the sentence "Do more with less." While it is a very minimalistic text editor, with the help of it's plugins and key bindings, it can become a powerfull and feature packed text editor.

  4. IDLE

    5. IDLE is the a text editor developed for Python itself. While it is so powerful for Python language, I personally doesn't like it a lot because of it's colorscheme.

  5. Sublime Text 2 and 3

    6. Sublime Text 2 and 3 is the text editor I preferred once for using Python development instead of IDLE, because while it is as powerful as IDLE, it also have a great color scheme.

  6. Android Studio

    7. I use Android Studio in order to develop Android applications with Java.

  7. Sonic Pi

    8. Sonic Pi is a live music coding environment where it allows people to develop algorithmic music, in other words algorave, by programming.
